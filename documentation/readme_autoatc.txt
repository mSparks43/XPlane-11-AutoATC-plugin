AutoATC build 0.9.7.1

INSTALL:
simply extract the AutoATC folder to X-Plane/Resources/plugins

Windows users:

additionally install the required support libraries:
VC_redist.x64.exe (MSVC VC++ 2019) - required for multithreading support.
vcredist_x64.exe (MSVC VC++ 2010 SP1) - required for plugin support

UPDATING:
Delete any previous AutoATC files
extract the AutoATC folder to X-Plane/Resources/plugins
run vcredist_x64.exe
